1. AI-Driven Interaction Flow

เส้นทางการทำงานหลักถูกออกแบบให้ขับเคลื่อนด้วย Chat Interface แทนเมนูแบบดั้งเดิม ระบบสามารถตีความข้อความทั่วไป (Natural Language) เพื่อค้นหาฐานข้อมูล ข้อมูล SDS และสารเคมีต่างๆ แล้วตอบรับในรูปแบบ Rich UI Cards ทันที

  • รองรับการค้นหาด้วย CAS No, ชื่อสาร ฯลฯ
  • เชื่อมโยงโมดูล 3D & ชุดค้นหาแผนที่ Lab อัตโนมัติ

2. Inventory Lifecycle Flow

การบริหารจัดการวงจรชีวิตของจุดจัดเก็บ (Inventory Lifecycle) เต็มรูปแบบ ตั้งแต่การนำสารเคมีเข้าคลังกลาง (Stock In) การเบิกจ่าย หรือการยืม-คืน มีการอัปเดตสถานะแบบ Real-time พร้อมผูกติดกับแผนที่ห้อง Lab อย่างแม่นยำ

  • ระบบแจ้งเตือนสารเคมีใกล้หมด หรือถึงคราวหมดอายุ
  • ตรวจสอบตู้คอนเทนเนอร์เก็บสารได้อย่างละเอียด

3. Security & Admin Flow

กลไกการรักษาความปลอดภัยด้วยระบบ Permission Gate ที่จำกัดการเข้าสู่หน้าจอ จัดการสต็อกและการเข้าหน้าควบคุมระบบ (Admin Panel) โดยมี Session Token ควบคุม และจัดเก็บ Logs ทุก Activity ของผู้ใช้งาน

  • ป้องกันการเข้าถึงโมดูล Admin สำหรับผู้ใช้ทั่วไปเต็มรูปแบบ
  • Report Generator (CSV/PDF) จากฐานข้อมูลกลาง

03. การจำแนกสิทธิการใช้งาน (RBAC)

ROLE: ADMIN

ผู้ดูแลระบบ

  • สิทธิสูงสุดในระบบ (Superuser)
  • จัดการบัญชีผู้ใช้งานและกำหนดสิทธิ
  • ควบคุมและตั้งค่าระบบทั้งหมด
  • ตรวจสอบบันทึกข้อมูลและ System Logs
ROLE: CEO

ผู้บริหารระดับสูง

  • ดูรายงาน Dashboard และสถิติภาพรวมคลัง
  • ดูข้อมูลการลงทุนและภาพรวมระบบทั้งหมด
  • ตรวจสอบบันทึกการเบิก-จ่ายสารเคมี
  • ไม่สามารถจัดการข้อมูลผู้ใช้งานระดับลึก
ROLE: LAB

เจ้าหน้าที่ห้องปฏิบัติการ

  • จัดการรายการสารเคมี (นำเข้า/เบิกจ่าย)
  • ใช้งาน AI Assistant และค้นหาข้อมูลเคมี
  • ดูและดาวน์โหลดเอกสาร SDS
  • เปิดดูตำแหน่งสารเคมีบนแผนที่ 3D/AR
  • ไม่สามารถเข้าถึงเมนูผู้ดูแลระบบ

FEATURE 02

Inventory & Dashboard

แดชบอร์ดสรุปภาพรวมสำหรับ Lab Manager และ Admin แสดงสถิติคลังสารเคมีแบบ Real-time พร้อมกราฟวิเคราะห์แนวโน้มการใช้งาน รวมถึงระบบแจ้งเตือนระดับสต็อกต่ำและใกล้หมดอายุ

FEATURE 04

Mobile QR Scanner

เน้นการออกแบบสำหรับทำงานบนหน้างานจริง (Mobile-First Design) เจ้าหน้าที่สามารถนำสมาร์ทโฟนเปิดกล้อง สแกน QR/Barcode ที่ขวดหรือหน้าตู้สารเคมี ระบบจะแสดง Pop-up ดึงข้อมูลขึ้นมาให้ดู ยืนยัน หรือตรวจสอบเพื่อทำรายการเบิกจ่ายได้ทันที

  • แสดงผล Responsive เต็มจอแบบ Native App
  • สแกนรหัสฉับไว ดึงข้อมูลรวดเร็ว
  • ทำรายการยืม-คืน หน้าคลังได้ทันที
